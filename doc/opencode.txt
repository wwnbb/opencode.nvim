1. INTRODUCTION                                                *opencode-intro*

OpenCode.nvim provides a native Neovim experience for interacting with
OpenCode, the open source AI coding agent.

Features:
- Chat interface with markdown rendering
- Diff viewer for accumulated changes
- Lualine integration
- Command palette
- Permission handling

================================================================================
2. INSTALLATION                                              *opencode-install*

Using lazy.nvim: >
  {
    "wwnbb/opencode.nvim",
    dependencies = {
      "nvim-lua/plenary.nvim",
      "MunifTanjim/nui.nvim",
    },
    config = function()
      require("opencode").setup()
    end,
  }
<

================================================================================
3. CONFIGURATION                                              *opencode-config*

Basic setup: >
  require("opencode").setup({
    server = {
      host = "localhost",
      port = 9099,
    },
  })
<

================================================================================
4. COMMANDS                                                  *opencode-commands*

|:OpenCode|          Open chat window
|:OpenCodeToggle|    Toggle chat window
|:OpenCodeClose|     Close chat window

================================================================================
5. API                                                            *opencode-api*

Lua API functions: >
  local opencode = require("opencode")
  
  opencode.setup(opts)      -- Configure plugin
  opencode.toggle()         -- Toggle chat
  opencode.open()           -- Open chat
  opencode.close()          -- Close chat
<
